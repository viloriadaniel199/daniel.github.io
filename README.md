# daniel.github.io

Desarrollé y personalicé este sitio web utilizando herramientas de inteligencia artificial para acelerar el proceso de desarrollo. Adapté el diseño, la estructura, la experiencia de usuario y la funcionalidad a los requisitos específicos del proyecto, realizando ajustes manuales en el código y la configuración final.

##Objetivo del proyecto

Este proyecto fue desarrollado como una prueba práctica para explorar mis capacidades en diseño web, integración de herramientas de IA y personalización de interfaces. El objetivo principal fue crear una página funcional y profesional mientras experimentaba con nuevas tecnologías y flujos de trabajo.
